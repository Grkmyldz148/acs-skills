## Pick or compose an archetype

Open `rules/archetype-list.md` once. Pick the one that best matches
the inventory note's aesthetic — or pick two and **blend** them.
Then **depart from the template** based on the token sheet.

### How to use the list

1. Read `archetype-list.md` and find the row that matches your
   project. Common matches:
   - Terminal / CLI / dev-tool → `terminal-crt`
   - Newsroom, magazine, longform → `editorial-paper`
   - Bank, trading desk, payments → `precision-metal`
   - Art studio, agency, creative tool → `organic-wood`
   - Anti-design, raw, manifesto → `brutalist-raw`
   - High-end goods, watchmaker, hotel → `crystalline-luxury`
   - Indie game / playful product → `arcade-warm`
   - Medical, scientific, legal → `clinical-quiet`

2. **Don't copy the archetype's parameters.** It says "fundamentals
   80–200 Hz, 3–5 kHz click, dry room"; that is a *direction*, not
   a setting. Use your token sheet to land on specific numbers
   inside (or outside) that range.

3. **Blend when needed.** A modern dev-tool that's also playful
   blends `terminal-crt` and `arcade-warm`. Note both rows and
   take 1–2 traits from each.

### What you produce

A 1–2 sentence statement: which archetype(s), and what specific
deviations the tokens demand.

Example for Caret (terminal-aesthetic CLI tool):

> Archetype: `terminal-crt`, but with the dry/snappy bias of
> `precision-metal` because the page has zero shadow scale and
> `transition-duration: 100ms`. CRT phosphor decay is the
> identity element; the rest is dry.

This becomes the soul of the soundscape. Every preset you author
should be testable against it ("does this sound like a terminal
with snap?").

### Anti-pattern

Picking an archetype because it sounds cool, not because it
matches the project. A luxury watchmaker's site does not get
`arcade-warm` no matter how charming arpeggios are. The
inventory note overrules personal taste.
